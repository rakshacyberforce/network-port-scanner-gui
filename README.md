# 💀 Network Port Scanner (GUI)

A powerful GUI-based Network Port Scanner built with Python and Tkinter.
Designed with a modern hacker-style interface and enhanced scanning features.

## 🚀 Features

* 🔍 Scan open ports on a target IP / hostname
* 🎨 Dark Hacker UI (Black + Green Theme)
* 📊 Real-time Progress Bar
* ⚡ Multi-threaded fast scanning
* 📈 Displays Total Open Ports
* ❌ Error Handling (Invalid IP / Ports)
* 🖥️ Clean and User-Friendly Interface

---

## 🧠 How It Works

The tool uses Python's `socket` module to attempt TCP connections on specified ports.
If a connection is successful, the port is marked as **OPEN**.
